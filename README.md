# mapred_engine

1) git checkout the repo

2) install venv with:

    >> python3 -m venv env

3) activate the environment

    >> source env/bin/activate

4) run setup.py

    >> pip install -e .

5) Run the scripts

    >> python word_counter.py data/raw/if-kipling.txt

    >> python average_ratings.py data/raw/ratings.txt
